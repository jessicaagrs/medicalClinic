## Descrição do projeto :pencil:

<p align="justify">
  Web site de cadastro de clínicas, usuários e especialistas.
</p>

## Funcionalidades :wrench:

:heavy_check_mark: Login via senha ou google.

:heavy_check_mark: Cadastro de clínicas.

:heavy_check_mark: Painel de admin para clínicas com cadastro de especialistas.

:heavy_check_mark: Cadastro de usuários.

:heavy_check_mark: Cadastro de depoimentos de usuários.

:heavy_check_mark: Painel para usuários efetuarem agendamentos.

```
yarn ci

yarn dev

```
